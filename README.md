# jb-breakbeat-generator

James Brown Breakbeat Generator! A one-page homage to the Godfather of Soul, built with Vue 3 / Nuxt 3.

Press **PLAY** to spin up a random breakbeat with a matching gif on the TV. Click the record ("HIT ME!") to jump to another random breakbeat + gif combo.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
